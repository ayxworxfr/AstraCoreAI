# 什么是大语言模型（LLM）

**大语言模型**（Large Language Model，**LLM**）是基于 Transformer 架构、经过海量文本数据预训练的神经网络模型。

## 代表性模型

| 公司 | 模型系列 |
|------|----------|
| OpenAI | GPT-3.5 / GPT-4 / GPT-4o |
| Anthropic | Claude 3 / Claude 3.5 |
| Google | Gemini 1.5 / Gemini 2.0 |
| Meta | LLaMA 2 / LLaMA 3 |

## 核心能力

- 对话问答
- 文本摘要与翻译
- 代码生成与调试
- 内容创作

## 规模效应

模型参数量通常在 **数十亿到数千亿**之间。参数越多，模型的推理、理解和生成能力通常越强，但推理成本也越高。
