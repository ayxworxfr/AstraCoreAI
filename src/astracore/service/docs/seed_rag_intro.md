# 什么是 RAG（检索增强生成）

**检索增强生成**（Retrieval-Augmented Generation，**RAG**）是将信息检索与语言生成相结合的 AI 技术架构。

## 工作流程

```
用户问题
   │
   ▼
[Embedding 模型] → 问题向量
   │
   ▼
[向量数据库] → 检索 Top-K 相关文档片段
   │
   ▼
[LLM] ← 问题 + 检索到的上下文
   │
   ▼
最终回答（附带来源引用）
```

## 核心优势

- **缓解幻觉**：回答基于真实文档，而非模型"记忆"
- **知识可更新**：无需重新训练模型，更新文档库即可
- **可追溯性**：能标注引用来源

## 适用场景

企业知识库问答、客服机器人、文档智能检索等。
