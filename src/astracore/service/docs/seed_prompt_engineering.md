# 什么是提示词工程（Prompt Engineering）

**提示词工程**是指设计和优化输入给 LLM 的提示词，以引导模型输出符合预期的结果。

## 常用技巧

### 1. 零样本提示（Zero-shot）

直接描述任务，不给示例：

```
将以下文本翻译成英文：你好，世界。
```

### 2. 少样本提示（Few-shot）

提供 2-5 个示例引导模型：

```
正面 → positive
负面 → negative
中性 → ?
```

### 3. 思维链（Chain-of-Thought，CoT）

要求模型逐步推理：

```
请一步一步思考后再给出答案。
```

### 4. 角色扮演（Role Prompting）

```
你是一位资深 Python 工程师，请帮我审查以下代码...
```

## 重要性

良好的提示词设计能显著提升模型的**准确性**、**一致性**和**可控性**，是 AI 应用开发中不可或缺的技能。
