# 什么是 AI Agent（智能体）

**AI Agent**（智能体）是具备自主规划和工具调用能力的 AI 系统。

## 与普通 LLM 的区别

| 能力 | 普通 LLM | AI Agent |
|------|----------|----------|
| 多轮对话 | ✅ | ✅ |
| 工具调用 | ❌ | ✅ |
| 自主规划 | ❌ | ✅ |
| 持续执行 | ❌ | ✅ |

## 常见工具类型

- **搜索引擎**：获取最新信息
- **代码执行器**：运行 Python/JS 代码
- **文件系统**：读写本地文件
- **API 调用**：访问外部服务

## 主流框架

- **LangGraph**：基于图结构的状态机 Agent
- **AutoGen**：微软出品的多 Agent 框架
- **CrewAI**：角色化多 Agent 协作

## Multi-Agent 系统

由多个专职 Agent 协作完成复杂任务，典型角色划分：

- **规划者**：分解任务、制定计划
- **执行者**：调用工具完成具体步骤
- **评审者**：检验结果质量
