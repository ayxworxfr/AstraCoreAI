# 什么是 Embedding（文本嵌入）

**Embedding**（嵌入）是将文本、图像等高维离散数据映射到连续低维向量空间的技术。

## 核心原理

语义相似的文本 → 距离相近的向量

```
"猫是动物"  → [0.12, -0.34, 0.89, ...]
"狗是宠物"  → [0.15, -0.31, 0.85, ...]  ← 距离近（语义相似）
"量子力学"  → [-0.72, 0.55, -0.11, ...] ← 距离远（语义不同）
```

## 常用模型

| 模型 | 维度 | 特点 |
|------|------|------|
| `all-MiniLM-L6-v2` | 384 | 轻量快速，本地可运行 |
| `text-embedding-ada-002` | 1536 | OpenAI，效果好 |
| `text-embedding-3-small` | 1536 | OpenAI 新版，性价比高 |

## 在 RAG 中的作用

```
索引阶段：文档文本 → Embedding → 向量 → 存入向量数据库
检索阶段：查询文本 → Embedding → 向量 → 余弦相似度计算 → Top-K 结果
```

Embedding 质量直接决定 RAG 检索的准确性。
