# 什么是向量数据库

**向量数据库**是专门用于存储和检索高维向量数据的数据库系统。

## 工作原理

文本、图像等非结构化数据经过 **Embedding 模型**转换后得到稠密向量，向量数据库通过**近似最近邻（ANN）**算法实现高效的相似度搜索。

## 主流向量数据库对比

| 数据库 | 特点 | 适用场景 |
|--------|------|----------|
| **ChromaDB** | 轻量级、开源、本地优先 | 本地开发、中小规模 |
| **Pinecone** | 全托管云服务 | 生产环境、大规模 |
| **Weaviate** | 多模态支持 | 图文混合检索 |
| **Milvus** | 高性能、分布式 | 亿级向量规模 |
| **Qdrant** | Rust 编写、高性能 | 生产级部署 |

## 核心操作

- `upsert`：插入或更新向量
- `query`：按相似度检索 Top-K
- `delete`：删除指定向量
